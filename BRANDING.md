# NomadCoin Branding Guide

## 🎨 Color Palette

### Primary Colors
| Color | Hex | Usage |
|-------|-----|-------|
| **Cyan** | `#00BCD4` | Primary actions, links, accents |
| **Purple** | `#7C4DFF` | Secondary accents, gradients |
| **Gold** | `#FFD700` | Rewards, earnings, highlights |

### Background Colors
| Color | Hex | Usage |
|-------|-----|-------|
| **Deep Navy** | `#0A0E27` | Primary background |
| **Dark Navy** | `#1A1F4E` | Secondary background |
| **Surface** | `#1E1E1E` | Card backgrounds |

### Status Colors
| Color | Hex | Usage |
|-------|-----|-------|
| **Green** | `#4CAF50` | Success, mining active |
| **Red** | `#F44336` | Errors, stop mining |
| **Orange** | `#FF9800` | Offline mode, warnings |
| **Gray** | `#9E9E9E` | Inactive states |

## 🏷️ Typography

### Headings
- Font: System default (SF Pro on macOS, Roboto on Android)
- Weight: Bold (700)
- Color: White `#FFFFFF`

### Body Text
- Font: System default
- Weight: Regular (400)
- Color: Light gray `#E0E0E0`

### Mono/Code
- Font: SF Mono, Roboto Mono, or monospace
- Color: Cyan `#00BCD4`

## 📐 Logo Usage

### Minimum Size
- Digital: 32x32px
- Print: 1 inch

### Clear Space
- Minimum clear space = height of "N" in logo

### Don'ts
- ❌ Don't stretch or distort
- ❌ Don't change colors
- ❌ Don't add effects
- ❌ Don't rotate

## 🎯 Brand Voice

### Tone
- Professional yet approachable
- Technical but not intimidating
- Community-focused
- Forward-thinking

### Key Messages
- "Built for nomads, by nomads"
- "Mine anywhere, earn everywhere"
- "Your phone is your bank"
- "Offline when you need it"

## 📱 App Icon Specifications

### iOS
- 1024x1024px PNG
- No transparency
- Rounded corners handled by system

### Android
- Adaptive icon with foreground/background layers
- 108x108dp foreground
- 108x108dp background

### Desktop
- 512x512px PNG
- macOS: ICNS format
- Windows: ICO format
- Linux: PNG + SVG
