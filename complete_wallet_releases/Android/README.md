# NomadCoin Android Wallet v1.0.0

## Installation

### Method 1: Direct APK Installation (Non-Store)
1. Download `NomadCoin-Wallet-v1.0.0.apk`
2. Transfer to your Android device
3. Open file manager, navigate to Downloads
4. Tap the APK file
5. Tap "Install"
6. Grant permissions when prompted

### Method 2: Store Installation (Recommended)
- Coming to Google Play Store soon
- Stay tuned for store availability

## Features
- 📱 Mobile-first wallet design
- 💰 Create and import wallets
- 📤 Send transactions
- 📥 Receive with QR codes
- 📜 View transaction history
- 🔒 Encrypted private key storage
- ⚡ Real-time balance updates

## Requirements
- Android 8.0 or higher
- 50MB free storage
- Internet connection (optional for offline mode)

## Security
- Private keys encrypted locally
- No cloud storage
- Ed25519 signature verification
- BLAKE3 hashing

## First Run
1. Create a new wallet or import existing
2. Set up password for encryption
3. Backup your private key
4. Start transacting!
