### Manual Build (Developers)
For developers who want to build from source:

```bash
cd nomad_mobile_wallet
flutter build ios --release
```

Requirements:
- Xcode 14.0+
- iOS 12.0+ device/simulator
- Apple Developer account (for device testing)
- CocoaPods package manager

## Features
- 📱 Native iOS design
- 💰 Full wallet management
- 📤 Send transactions
- 📥 Receive with QR codes
- 🔐 Biometric security (Face ID / Touch ID)
- ⚡ Offline transaction support
- 📊 Real-time balance

## Security
- Private keys encrypted with AES-256
- Biometric protection
- No cloud storage
- Ed25519 cryptography

## System Requirements
- iOS 12.0 or higher
- iPhone 6s or newer
- 100MB free storage
